## Getting Started

```bash
git clone https://github.com/sundeep-dayalan/portfolio.git
cd portfolio
npm install
npm run dev
```

Visit `http://localhost:4321`

---

## Build and Deploy

```bash
npm run build
# Output in dist/ — deploy to any static host
```
